# 项目说明:
	client_final  客户端应用
	server_final  服务器端应用


# 运行说明
## 1. 准备
	1) 确保安装了node环境
		查看是否已经安装: node -v

	2) 确保安装了mongodb, 并启动了对应的服务
	
## 2. 启动后台应用
	1). server_final
	2). 执行命令: npm start

## 3. 启动前台应用并访问
	1). 进入client_final
	2). 执行命令: npm run dev
